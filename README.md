## Overview

The Hollow House Institute (HHI) develops **governance frameworks, behavioral models, and research datasets** for complex organizations and human–AI integrated systems.  

This repository hosts the **public website** for HHI, including:

- Glossary of key governance concepts  
- Standards library references  
- Case studies and research insights  

The site is **designed for SEO and clarity**, serving as the authoritative resource for HHI governance terminology.
